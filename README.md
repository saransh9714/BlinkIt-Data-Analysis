# 🛒 Blinkit Retail Data Analysis

## 📌 Project Overview
Blinkit Retail Analysis is a data-driven project focused on extracting actionable business insights from Blinkit's retail transaction dataset.  
The project involves data cleaning, transformation, exploratory data analysis (EDA), KPI computation, and visualization to understand sales patterns, customer behavior, and outlet performance.

This project demonstrates end-to-end data analytics workflow using Python.

---

## 🎯 Objectives
- Clean and preprocess raw retail data
- Analyze sales performance across product categories and outlets
- Identify revenue trends and customer preferences
- Compute key business KPIs
- Generate visual insights for strategic decision-making

---

## 📊 Dataset Details
- 85,000+ transaction records
- 6 Product Categories
- 3 Fat Content Types
- 45 Regions / Outlet Locations
- Customer Ratings & Sales Data

---

## 🛠️ Tech Stack
- **Python**
- **Pandas** – Data Cleaning & Manipulation
- **NumPy** – Numerical Computation
- **Matplotlib / Seaborn** – Data Visualization
- **SQL (Optional)** – KPI & Aggregation Queries
- **Jupyter Notebook**

---

## 📈 Key Analysis Performed

### 🔹 Sales Analysis
- Total Sales Revenue

### 🔹 Outlet Analysis
- Outlet Location Performance
- Outlet Size vs Revenue
- Tier-wise Sales Comparison

### 🔹 Customer Insights
- Average Customer Ratings
- Most Preferred Product Categories
- Sales Trends by Region
